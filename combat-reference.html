Clash Rules

  • Combat takes place in a dedicated modal above the map and roster page.
  • Each clash has five phases: Selection, Reveal, Reaction, Calculation, Activation.
  • Each card has one printed attack lane and one printed defense lane.
  • If an attack lane matches a defense lane, that strike is blocked.
  • If it does not match, the defender loses 1 Hit Point.
  • At 0 Hit Points, that fighter loses, becomes Injured, loses 1 Reputation, and returns to the main game.
  • Hit Points and Form Points reset after the fight ends.

Modes

  • School Special: use the fighter keyword on that clash. Cost: 1 Form Point.
  • Swap Attack: replace the printed attack lane with the card's swap lane. Cost: 1 Form Point.
  • Swap Defense: replace the printed defense lane with the card's swap lane. Cost: 1 Form Point.
  • The special 10th card matches the live app: keyword active for 0 cost, or pay 1 Form Point for one swap.

    Deck Flow

    • At combat start, each fighter draws 5 cards.
    • Clashes can only use cards currently in hand.
    • Used cards leave the hand until reshuffle.
    • When the first 5 are gone, draw the second 5.
    • If all 10 are exhausted, reshuffle and draw again so combat always continues.
